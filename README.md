# Activity Tracker

A Python-based activity tracking application for monitoring user activity patterns.

## Features

- Activity tracking and monitoring
- Session analytics
- Configurable tracking parameters
- Human-like behavior simulation

## Requirements

- Python 3.x
- See `requirements.txt` for Python dependencies

## Installation

1. Clone this repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the activity tracker:
```bash
python simple_activity_tracker.py
```

## Configuration

Edit `tracker_config.json` to customize tracking parameters.
